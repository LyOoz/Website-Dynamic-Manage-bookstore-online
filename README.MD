# Dynamic Online Bookstore
A full-stack dynamic web application for managing an online bookstore.
Designed for scalability, clean structure, and real-world usage.

# Features
- Browse books dynamically from database
- Search and filter functionality
- Shopping cart system
- User authentication (login/register)
- Order management system
- Admin panel for managing books & users
- Dynamic content rendering (no static pages)

# Tech Stack

- Frontend: HTML,CSS,JavaScript
- Backend: PHP
- Database: MySQL

# Installation
1. Clone the repository
``` bash
git clone https://github.com/LyOoz/Website-Dynamic-Manage-bookstore-online
```
2. Move project to your server environment (e.g. XAMPP / Node server)
3. Setup database
- Import the .sql file into your database (.sql in folder db)
- Update database config in project files
4. Start the server
``` bash
npm start
# or
php -S localhost:8000
```
